# Headless Horseman

Dépose ici le PDF de la fiche personnage du Headless Horseman (ex: `headless_horseman.pdf`).

Claude Code pourra le lire avec le tool `Read` pour comprendre les mécaniques du personnage
avant d'implémenter ou modifier des capacités.

## Convention pour les autres personnages

Chaque personnage a son propre sous-dossier dans `characters/` :
```
characters/
  headless_horseman/
    headless_horseman.pdf
  moon_elf/
    moon_elf.pdf
  ...
```
