# \# Headless Horseman — Dice Throne Strategy Guide

# 

# \## Hero Overview

# Headless Horseman is an aggro hero that seeks to synchronize his Haunted Head with his Dreadful tokens to improve the power of his defense, set up bursts of damage with Terrorize, and, with a bit of luck, slam Spectral Assault for absurd amounts of value.

# 

# \*\*Dice anatomy:\*\* AAABBC  

# \- Face 1, 2, 3 = \*\*A\*\* = Axe  

# \- Face 4, 5 = \*\*B\*\* = Horseshoe  

# \- Face 6 = \*\*C\*\* = Scare (pumpkin)

# 

# \## Dreadful Tokens

# \- Gagner le \*\*4e Dreadful\*\* déclenche \*\*Terrorize\*\* : 3 dmg + 1 CP.  

# \- "The fifth Dreadful is only really worth about half a Dreadful because it removes the ability to gain Dreadful on the defense."  

# \- On évite de gagner le 5e avec une Offensive Ability (ça veut dire que l’ability était sous-performante).

# 

# \## Haunted Head Mechanic

# \- Commence sur ton board au début de la partie.  

# \- Quand tu jettes le Head à l’adversaire : +1 Dreadful maintenant + 1 à la fin du tour.  

# \- Quand l’adversaire l’a : +1 Dreadful à la fin de ton upkeep.  

# \- Bonus sur \*\*Reap\*\* et \*\*Horrify\*\* uniquement si tu as le Head.

# 

# > Stratégie générale : Tu balances le Head entre toi et l’adversaire jusqu’à \~3 Dreadful. Après, tu le laisses souvent chez l’adversaire pour éviter les awkwardness.

# \### Stratégie Dreadful Count (la plus importante)

# \- 0-1 Dreadful → Priorité \*\*Reap\*\*  

# \- 2 Dreadful → \*\*Reap\*\* excellent  

# \- 3 Dreadful → \*\*Small Straight\*\* ou \*\*Ride Down\*\*  

# \- 4-5 Dreadful → \*\*Ride Down\*\* devient le meilleur move  

# → Toujours garder un œil sur \*\*Spectral Assault\*\* quoi qu’il arrive.
